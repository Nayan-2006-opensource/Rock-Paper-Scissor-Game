Rock Paper Scissors Game

A simple Rock Paper Scissors game built using Python.

Features

* Play against the computer.
* Random computer moves using the random module.
* Input validation for invalid moves.
* Live score tracking.
* Game ends when either the user or the computer reaches 3 points.

Technologies Used

* Python

How to Run

1. Clone this repository.
2. Open the project folder.
3. Run the Python file:

python rock_paper_scissors.py
